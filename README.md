## Zoo projekt

Ett C# projekt för att simulera ett Zoo genom objektorientad programmering.

* Funktioner
	* Skapa olika burar i Zoo:et. Finns även en funktion som hämtar antalet burar i Zoo:et.
	* Fylla burarna med olika djur.
	* Varje djur är av en specifik art, som du kan namge :)
	* Ta reda på vilka djur som befinner sig i buren.
	* Vissa djur äter andra djur. Om dessa djur hamnar i samma bur så kommer rovdjuret att äta bytet. Om detta sker kommer programmet att berätta vilket byte som rovdjuret ätit upp.
